Follow these instructions, as given by Dr. Christopher Weaver at the University of Oklahoma.
For this project, you will need to have a version of Eclipse installed that has the BuildShip plugin installed.

Next, you will run gradlew elipse in the enter-the-cube directory.
In the Eclipse import wizaes, open enter-the-cube as a Gradle/Existing Gradle Project.

To compile the code, you can run it in eclipse by pressing Run Application button or by going to the
enter-the-cube directory in command line and typing gradlew installDist.
